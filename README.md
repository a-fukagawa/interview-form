# Interview Form RDF

LINC Interview Form RDF project

## How to contribute

For the first time:

```sh
% git clone https://github.com/med2rdf/interview-form.git
% cd interview-form
```

Afterwards:

```sh
% git pull
(edit benitas/sectionXX.ttl)
% git status
% git diff
% git add benitas/sectionXX.ttl
% git commit
(add comments on edits)
% git push
```
